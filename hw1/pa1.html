Part B: Theory Problem¶

1. From continuous thresholds to a finite class.¶

Grid Design: We define a finite grid $G \subseteq [0, 1]$ as follows: $$G = \{ k \cdot \Delta : k \in \mathbb{Z}, 0 \le k \cdot \Delta \le 1 \}$$ The size of this grid is $|G| = \lfloor 1/\Delta \rfloor + 1$, which depends only on $\Delta$.

Proof of Realizability: Let $((x_t, y_t))_{t=1}^T$ be a $\Delta$-separated threshold-realizable sequence. By definition, there exists $\theta^* \in [0, 1]$ such that $y_t = h_{\theta^*}(x_t)$ and $|x_t - \theta^*| \ge \Delta$ for all $t$.

Consider $\tilde{\theta} = \lfloor \theta^* / \Delta \rfloor \cdot \Delta$. Clearly $\tilde{\theta} \in G$ and $\tilde{\theta} \le \theta^* < \tilde{\theta} + \Delta$. We check if $h_{\tilde{\theta}}(x_t) = h_{\theta^*}(x_t)$ for all $t$:

  1. If $x_t \ge \theta^*$, then $y_t = +1$. Since $\tilde{\theta} \le \theta^*$, we have $x_t \ge \tilde{\theta}$, so $h_{\tilde{\theta}}(x_t) = +1$.
  2. If $x_t < \theta^*$, then $y_t = -1$. By the separation assumption, $x_t \le \theta^* - \Delta$. Since $\theta^* < \tilde{\theta} + \Delta$, we have $\theta^* - \Delta < \tilde{\theta}$. Thus $x_t < \tilde{\theta}$, so $h_{\tilde{\theta}}(x_t) = -1$.

In both cases, $h_{\tilde{\theta}}(x_t) = y_t$, so the sequence is realizable by $H_G$.

Mistake Bound: The Halving algorithm on a finite class $H$ makes at most $\log_2(|H|)$ mistakes. Here $|H_G| = |G| \approx 1/\Delta$. The mistake bound is: $$M \le \log_2(|G|) = \log_2(\lfloor 1/\Delta \rfloor + 1) = O(\log(1/\Delta))$$

2. A positive margin from separation.¶

Feature Map and Representation: Choose the feature map $\phi(x) = [x, 1]^\top \in \mathbb{R}^2$. For a threshold $\theta^*$, define the weight vector $u^* = \frac{1}{\sqrt{1 + (\theta^*)^2}} [1, -\theta^*]^\top$. Then $\langle u^*, \phi(x) \rangle = \frac{x - \theta^*}{\sqrt{1 + (\theta^*)^2}}$. The prediction is $\text{sign}(\langle u^*, \phi(x) \rangle) = \text{sign}(x - \theta^*) = h_{\theta^*}(x)$.

Margin and Norm Bounds:

  1. Norm Bound: $\|\phi(x)\| = \sqrt{x^2 + 1}$. Since $x \in [0, 1]$, $\|\phi(x)\| \le \sqrt{2}$. Thus $R = \sqrt{2}$.
  2. Margin Bound: The functional margin is $y_t \langle u^*, \phi(x_t) \rangle = \frac{y_t(x_t - \theta^*)}{\sqrt{1 + (\theta^*)^2}} = \frac{|x_t - \theta^*|}{\sqrt{1 + (\theta^*)^2}}$. Since $|x_t - \theta^*| \ge \Delta$ and $\sqrt{1 + (\theta^*)^2} \le \sqrt{2}$, the margin $\gamma \ge \frac{\Delta}{\sqrt{2}}$. This satisfies $\gamma \ge c\Delta$ with $c = 1/\sqrt{2}$. Note that $\lVert u^∗ \rVert = 1$ so the functional and geometric margins coincide.

Mistake Bound: By Novikoff's Theorem, the Perceptron makes at most $M \le \frac{R^2}{\gamma^2}$ mistakes. Substituting $R^2 = 2$ and $\gamma^2 \ge \Delta^2/2$: $M \le \frac{2}{\Delta^2/2} = \frac{4}{\Delta^2} = O(1/\Delta^2)$.

3. Comparison and interpretation.¶

Continuous-Threshold Impossibility: The impossibility result for continuous thresholds states that without any separation assumption, the mistake bound is infinite because an adversary can always pick a point arbitrarily close to the current threshold. Part 1 does not contradict this because the $\Delta$-separation assumption explicitly forbids points in the interval $(\theta^* - \Delta, \theta^* + \Delta)$. This gap allows us to discretize the space into a finite grid without losing the true classifier, effectively reducing the infinite class to a finite one.

Comparison of Bounds:

  1. Halving (Part 1): $O(\log(1/\Delta))$.
  2. Perceptron (Part 2): $O(1/\Delta^2)$.

Interpretation: The bounds scale differently because they utilize different properties of the hypothesis class. The Halving bound is information-theoretic; it depends on the "size" (complexity) of the hypothesis space. By discretizing, we show the effective complexity is logarithmic in $1/\Delta$. The Perceptron bound is geometric; it depends on the margin between classes in a specific embedding. The $1/\Delta^2$ scaling is characteristic of linear learners that only see the geometry of the points and do not exploit the specific 1D structure of thresholds as efficiently as the Halving algorithm.

4. Optional strengthening: audit an AI proof.¶

Mathematical Problems:

  1. Incorrect Bound for Perceptron: The argument claims Perceptron must make $O(\log(1/\Delta))$ mistakes. However, the Perceptron mistake bound is $R^2/\gamma^2$. As shown in Part 2, for the standard feature map, this yields $O(1/\Delta^2)$. Perceptron is a linear learner and does not achieve the logarithmic bound of the Halving algorithm without a more complex (possibly infinite-dimensional) feature map.
  2. Conflation of Class Size and Learner Performance: The argument assumes that because the "effective" class size is $O(1/\Delta)$, any learner makes $O(\log(1/\Delta))$ mistakes. This is false; the $O(\log |H|)$ bound applies to the Halving algorithm (and similar optimal learners), but not to all online learners. Different algorithms have different mistake bounds for the same class.

Corrected Statement: "For every $\Delta$-separated threshold-realizable sequence, there exists a finite sub-class of thresholds $H_G$ of size $O(1/\Delta)$ that contains a consistent hypothesis. Consequently, the Halving algorithm achieves a mistake bound of $O(\log(1/\Delta))$, which is the optimal mistake bound for this class."

Proof: As proven in Part 1, the grid $G$ with spacing $\Delta$ creates a finite class $H_G$ with $|H_G| \le 1/\Delta + 1$. The Halving algorithm maintains a version space of consistent hypotheses and predicts by majority vote. Each mistake reduces the version space by at least half. Since the true threshold (or its grid representative) is in $H_G$, the process must terminate within $\log_2 |H_G| = \log_2(O(1/\Delta)) = O(\log(1/\Delta))$ mistakes.

Part C: Perceptron — Implementation and Experiments¶

Algorithm:
Input: Feature map $\phi: \mathcal{X} \to \mathbb{R}^d$, labels $y_t \in \{-1, +1\}$

  1. Initialize $w_1 = \mathbf{0}$
  2. Receive $x_t$, predict $\hat{y}_t = +1$ if $\langle w_t, \phi(x_t) \rangle \geq 0$, else $\hat{y}_t = -1$
  3. Receive true label $y_t$
  4. If $\hat{y}_t \neq y_t$ (mistake): $w_{t+1} = w_t + y_t \cdot \phi(x_t)$
  5. Else: $w_{t+1} = w_t$

Mistake Bound (Novikoff's Theorem): $M \leq \dfrac{R^2}{\gamma^2}$

where $\gamma = \min_t y_t \langle w^*, \phi(x_t) \rangle$ (margin) and $R = \max_t \|\phi(x_t)\|$ (radius).

def perceptron(X, y):
    """
    Online Perceptron algorithm.

    Parameters
    ----------
    X : ndarray, shape (T, d)  -- feature vectors phi(x_t)
    y : ndarray, shape (T,)    -- labels in {-1, +1}

    Returns
    -------
    w             : final weight vector
    mistakes      : total number of mistakes
    mistake_times : list of time-steps where a mistake occurred
    """
    T, d = X.shape
    w = np.zeros(d)
    mistakes = 0
    mistake_times = []

    for t in range(T):
        y_hat = 1 if np.dot(w, X[t]) >= 0 else -1
        if y_hat != y[t]:          # mistake
            w = w + y[t] * X[t]
            mistakes += 1
            mistake_times.append(t)

    return w, mistakes, mistake_times

1. How do you generate data where you know $\gamma$ and $R$?¶

Strategy:

  1. Choose a ground-truth separator $w^* = e_1 \in \mathbb{R}^d$ (first standard basis vector, unit norm).
  2. Sample raw points $z_t \sim \mathcal{N}(0, I_d)$.
  3. Assign labels $y_t = \text{sign}(\langle w^*, z_t \rangle)$.
  4. Enforce margin $\gamma$: shift each $z_t$ along $w^*$ so that $y_t \langle w^*, z_t \rangle \geq \gamma$: $$\phi(x_t) = z_t + y_t \cdot \max(0,\, \gamma - y_t \langle w^*, z_t \rangle) \cdot w^*$$
  5. Enforce radius $R$: normalise each point to the sphere of radius $R$: $$\phi(x_t) \leftarrow R \cdot \frac{\phi(x_t)}{\|\phi(x_t)\|}$$

Choices required:

  • Dimension $d$, number of examples $T$
  • The separator $w^*$ (we use $e_1$)
  • Whether to fix $\|\phi(x_t)\| = R$ exactly (sphere) or $\leq R$ (ball) -- we use the sphere
  • The pre-rescaling margin target; the actual margin after rescaling is measured empirically

def generate_data(T, d, gamma, R=1.0, seed=None):
    """
    Generate a linearly separable dataset with known margin gamma and radius R.

    Parameters
    ----------
    T            : number of examples
    d            : feature dimension
    gamma        : desired margin  (y_t <w*, phi(x_t)> >= gamma for all t)
    R            : desired radius  (||phi(x_t)|| == R for all t)
    seed         : random seed

    Returns
    -------
    X            : ndarray (T, d)  -- feature vectors
    y            : ndarray (T,)    -- labels in {-1, +1}
    w_star       : ndarray (d,)    -- ground-truth separator (unit vector)
    actual_gamma : float           -- actual minimum margin after rescaling
    """
    rng = np.random.default_rng(seed)

    # Ground-truth separator: first standard basis vector
    w_star = np.zeros(d)
    w_star[0] = 1.0

    # Sample raw Gaussian points
    Z = rng.standard_normal((T, d))

    # Assign labels from the separator
    scores = Z @ w_star                          # shape (T,)
    y = np.where(scores >= 0, 1, -1)

    # Enforce margin: shift along w_star so y_t * <w*, z_t> >= gamma
    functional_margins = y * scores
    deficit = np.maximum(0.0, gamma - functional_margins)
    X = Z + (deficit * y)[:, None] * w_star

    # Enforce radius: normalise to unit sphere, then scale by R
    norms = np.linalg.norm(X, axis=1, keepdims=True)
    X = X / norms * R

    actual_margins = y * (X @ w_star)
    actual_gamma = float(actual_margins.min())

    return X, y, w_star, actual_gamma

5. What happens as $\gamma \to 0$? Connection to the threshold counterexample.¶

As $\gamma \to 0$ the data approaches a non-separable regime. The bound $R^2/\gamma^2 \to \infty$, and in the limit the Perceptron may never converge.

Threshold counterexample: Consider $d=1$, $x_t = 1$ for all $t$, with labels alternating $+1, -1, +1, -1, \ldots$. The data is not linearly separable (no single threshold separates them), and the Perceptron makes a mistake on every single example -- $M = T$.

# Part B: Threshold counterexample
def threshold_counterexample(T):
    """1-D example: x_t = 1 always, labels alternate +1/-1."""
    X = np.ones((T, 1))
    y = np.array([1 if t % 2 == 0 else -1 for t in range(T)])
    return X, y

Observations:

  1. As $\gamma \to 0$: The number of mistakes $M$ grows rapidly (roughly as $1/\gamma^2$), and the Perceptron requires more and more rounds to converge. For very small $\gamma$, the bound $R^2/\gamma^2$ becomes enormous, and the algorithm may exhaust the training set before converging.

  2. Threshold counterexample ($\gamma = 0$): When the data is not linearly separable, the Perceptron makes a mistake on every example -- cumulative mistakes grow linearly with $T$. This is the worst case: $M = T$. The alternating-label example with a single feature $x_t = 1$ is the canonical demonstration that the Perceptron requires linear separability (positive margin) to have a finite mistake bound.

  3. Connection: The threshold counterexample is the $\gamma \to 0$ limit. As $\gamma$ decreases toward 0, the data becomes "almost non-separable," and the Perceptron's behaviour transitions from fast convergence to the divergent worst-case regime.
